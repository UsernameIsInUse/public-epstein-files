this repo stores no videos or content, it simply provides links to publicly accessible videos on the justice.gov website

this is provided as-is as a resource to easily access those public assets stored on government servers

viewer discretion is advised.

--

`videos.txt` has all of the raw urls scraped from justice.gov/epstein that were found in the search "no images produced". includes duplicates

`confirmed.txt` has all of the urls confirmed to be mp4s

`avis.txt` has all of the urls confirmed to be avis, some will download straight to your computer, others will play in browser

`unconfirmed.txt` has all of the urls which have not yet found a file type associated with it
